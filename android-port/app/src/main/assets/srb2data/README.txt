Place SRB2 game data files (srb2.pk3, zones.pk3, characters.pk3, music.pk3) 
here to bundle them into the APK.

Files placed here will be automatically copied to the game's data directory
on first launch.

Alternatively, you can push files to the device manually after installing:
  adb push srb2.pk3 /sdcard/Android/data/org.srb2.android/files/SRB2/
